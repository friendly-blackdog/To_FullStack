# Sckit-Learn을 활용한 MachineLearnig Basic
```
머신러닝을 AI관점의 한 분야인 AI-머신러닝-딥러닝측면에서 설명하기 보다는 DataScience 측면에서 설명하는 것을 지향으로 목차를 구성하였습니다.

예제용 DataSource는 유명한 kaggle문제를 활용할 예정입니다.
```
